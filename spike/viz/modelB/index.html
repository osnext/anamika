Splash screen

Representation of the internals of the system in a model easily understood by the user. This is a sample viz, could be modified to better represent the user's model. The key point is that it shows both the internals and the startup process: Green are loaded subsystems, orange are being loaded, white is to be loaded.

All subsystems are loaded

There is no login screen. System loads up a public desktop by default, and the default desktop shows recently used apps, content and tags

Pressing a system key zooms out to show all running apps

Pressing the same system key again zooms out to show all workspaces for the user

Pressing the same system key again zooms out to show all users in the system

Pressing the same system key again zooms out to show the system itself
